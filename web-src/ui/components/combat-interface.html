战斗系统说明

《诡秘之主》的战斗系统基于非凡能力对抗，核心机制：

  • 生命值：降为0则战斗失败
  • 灵性值：使用技能消耗，每回合自动恢复少量
  • 理智值：使用某些能力或受到精神攻击会降低，过低可能引发失控
  • 序列克制：不同途径存在克制关系，影响伤害效果
  • 状态效果：增益/减益效果持续数回合

战斗策略：根据敌人类型选择合适的技能序列，注意灵性管理和理智保持。
