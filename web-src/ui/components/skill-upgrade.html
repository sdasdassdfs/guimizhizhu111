分支效果对比

不同分支的战斗风格和适合场景

精准射击
  • 高命中率，适合对抗敏捷敌人
  • 稳定输出，暴击率提升
  • 射程优势，安全距离战斗
  • 推荐给喜欢控制战斗节奏的玩家
强力射击
  • 高爆发伤害，一击制敌
  • 可能击退或眩晕敌人
  • 消耗更多灵性值
  • 推荐给追求极致输出的玩家
多重射击
  • 同时攻击多个目标
  • 对付群体敌人效果显著
  • 单体伤害略有降低
  • 推荐给需要应对围攻的玩家
