初次占卜

简单 晋升任务 要求: 序列9

作为刚刚踏入非凡世界的新人，你需要掌握最基本的占卜能力。占卜是窥秘途径的核心技能之一，能够帮助你预见危险、寻找线索。

任务要求：

  1. 准备完整的占卜用具（仪式银匕、蜡烛、灵摆等）
  2. 选择一个明确的占卜问题（避免模糊不清的提问）
  3. 在灵性充沛的环境中进行占卜仪式
  4. 记录占卜结果并分析其准确性

提示：占卜的成功率受到灵性状态、环境因素和问题质量的影响。

任务目标

  • 准备占卜用具（蜡烛、灵摆、仪式银匕）
  • 选择合适的占卜问题
  • 进行一次有效的占卜仪式
